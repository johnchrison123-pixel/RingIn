# Friend-Finder & Discovery Apps — 2025-2026 Research

## Bumble BFF — 2026 state

**Discovery:** One-on-one swipe + gender, interests, location. Post-Sept 2025 relaunch: pivot to group discovery via acquired Geneva platform. Group discovery launched Feb 2026.

**Filter UX:** Gender, interests, location radius. Profile prompts as soft filters.

**Anti-burnout:** Shift from infinite swipe to group discovery. "Plan" tool schedules hangouts. Event calendar.

**Premium:** BFF Premium unlocks unlimited swipes, advanced filters, profile boost.

**What RingIn lacks vs this:**
- In-app event/hangout planning
- Interest-based community rooms
- Profile prompts beyond bio

## Yubo — 2026 state

**Discovery:** Profile cards with interest tags + location + age. No likes/follows (intentional). Live streams (up to 10 people) as primary discovery. Real-time AI moderation (Yoti liveness). 85M users.

**Anti-burnout:** No likes, no follower counts. Live room discovery replaces profile grinding. Age 18+ enforced Nov 2025.

**Premium:** YuBucks microtransaction. Revenue grew €7M (2019) → €25M+.

**What RingIn lacks:**
- Live group voice/video rooms as discovery surface
- Virtual gift economy for social interactions
- No-follower-count profile mode

## Litmatch — 2026 state

**Discovery:** AI Friend Discovery on personality, hobbies, preferences. Soul Game — text-based Q&A before mutual like. Party Chat group voice with mic slots.

**Filter UX:** Interest + language matching (strong in SEA). Party rooms topic-tagged.

**Premium:** Diamonds for gifts/boosts. VIP status. MOD APKs heavily searched (piracy pressure).

**What RingIn lacks:**
- Structured icebreaker game before connecting (Soul Game)
- Party Chat rooms for group voice discovery
- Voice-based gift/virtual currency loop

## Wink, Patook, Locket

**Wink:** Interest + age + location. Cross-platform username exchange (Snapchat). Gem economy.

**Patook:** Weighted point scoring (users assign weights). Flirt Detection AI blocks pick-up lines. Local Discussions tab. Explicitly platonic.

**Locket:** No algorithmic discovery — closed network (up to 20 friends). Home screen widget mechanic. 91M installs, 9M DAU, profitable on $12.5M raised. Scheduled rituals (Rollcall — weekly group selfies). 100,000+ paid subscribers.

**What RingIn lacks (cross-app):**
- Anti-creep / intent enforcement signal layer
- User-weighted compatibility scoring
- Local discussion board / community feed
- Scheduled ritual mechanic (weekly group check-in)
- Home screen / lock screen presence (widget API)

## Snap Map / Threads / Lemon8 / BeReal

**Snap Map:** 350M monthly users. Location-based discovery. Heat maps. "Location Legend" (2026). Family Center. Ghost Mode as escape valve.

**Threads:** Interest graph not social graph. 200+ topic Communities (Oct 2025). "Dear Algo" (2026) algo control. No public like counts by default.

**Lemon8:** ByteDance TikTok-derived FYP for lifestyle. US ceased Jan 2025 but active India/SEA/Japan.

**BeReal:** Acquisition €500M by Voodoo. >$30M revenue end 2025. Daily ritual proves time-limited rituals outperform infinite feeds.

**What RingIn lacks:**
- Location map view of nearby friends (opt-in Ghost Mode)
- Event/venue-based friend discovery
- Interest community rooms with discovery boost
- Daily ritual moment with time-limited ephemeral content
- Content-led discovery
- Early-creator algorithmic boost

## India context — FRND

285M+ conversations, 418M voice minutes in 2025. 92% engagement from outside Tier 1. 79% on mid/entry devices. Language-first matching. Avatar identity. Interest-based + nearby friend discovery.

## TOP 7 FEATURES TO ADD TO RINGIN'S FRIENDS NAV

| # | Feature | Impact | Build Cost |
|---|---------|--------|-----------|
| 1 | Live Group Voice Rooms as Discovery | Very High | Low (Agora exists) |
| 2 | Structured Icebreaker Game Before Connect (Soul Game) | High | Low |
| 3 | User-Weighted Compatibility Score (Transparent) | High | Low-Med |
| 4 | Daily Voice Ritual (BeReal × RingIn) | High | Med |
| 5 | Opt-in Proximity Map (Snap Map for RingIn) | High | Med |
| 6 | Interest Community Rooms | Very High | Med |
| 7 | Anti-Creep Intent Badge + Classifier | Med-High | Med-High |

### Detail on #1: Live Group Voice Rooms
Open rooms (max 8-10) tagged by interest. Any user joins/leaves. Voice-first. The room IS discovery. Yubo built 85M users on this. FRND 418M minutes. RingIn already has voice infra.

### Detail on #2: Icebreaker Before Connect
Before DM, 3-question game from pool ("coffee or chai?", "last song stuck in head?"). Both answer. If both like answers, connect. Skip available. Filters intent. Litmatch's #1 differentiator.

### Detail on #3: User-Weighted Compatibility
Users assign weights to filter dimensions. App computes score per profile, shows "87% compatible". Patook proved this outperforms ML black-box for trust.

### Detail on #4: Daily Voice Ritual
Once-daily notification. 15-second voice note or quick photo. Goes to Friends tab. Disappears 24h. Friends react with voice emoji. Locket proved 9M DAU; BeReal proved engagement.

### Detail on #5: Proximity Map
City-level (not precise) location opt-in. "Near You" section: users within X km online now. Ghost Mode default. 350M Snap Map users.

### Detail on #6: Interest Community Rooms
Named rooms (Bollywood, Startup Founders, Night Owls). Post text/voice inside. Active rooms surfaced. Joining = discoverable to members. Bumble BFF 2025 pivot. Threads 3x reach.

### Detail on #7: Anti-Creep Intent Badge
Profiles declare intent: "Study partner / Creative collaborator / Gaming friends / Language exchange / Local hangout." Badge shown. Optional AI classifier flags off-intent messages, warns sender. Patook brand.
